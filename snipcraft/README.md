# snipcraft
Web-Based Online Screen Recorder &amp; Editor

Check the Software Requirement Specification [here](docs/SRS.md)

## How to run the project

1. Clone the repository: use github desktop for easy cloning
2. Open the project in your favorite IDE (VS Code recommended)
3. Open the terminal in the IDE and run the following commands:
```bash
npm install
```
4. After the installation is complete, run the following command to start the development server:
```bash
npm run dev
```
